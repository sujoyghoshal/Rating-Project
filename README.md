# E-Commerce & Reviews Website

A modern, responsive e-commerce website built with **React** and **Vite**, showcasing products, categories, and user reviews. This project includes dummy data for categories, products, and reviews, making it ideal for frontend testing or prototyping.

## Features

- Dynamic product categories with icons
- User reviews with ratings
- Responsive design for mobile and desktop
- Search functionality for products and categories
- Header and footer with navigation links
- Signup button (frontend only)
- Easy customization of products, categories, and reviews

## Technologies Used

- **Frontend:** React, Vite, JSX, CSS
- **Data:** JSON for categories and reviews
- **Deployment:** Static website-ready
