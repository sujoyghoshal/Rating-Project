# 🌟 E-Commerce & Reviews Website

**Live Demo:** [https://demorating-one.vercel.app/](https://demorating-one.vercel.app/)  

A modern, responsive e-commerce and review website built using **React + Vite** for the frontend. The backend (upcoming) will use **Django** with an **SQL database** to store products, categories, reviews, and users.  

---

## 🚀 Features

- Dynamic product categories with icons  
- User reviews with ratings, company details, and images  
- Responsive design for all devices  
- Search functionality for categories and products  
- Header and footer with navigation links  
- Upcoming backend: Django + SQL database  
- Easy scalability with reusable React components  

---

## 🛠 Technologies Used

### Frontend
- React (JSX)  
- Vite  
- CSS  
- JSON for dummy data (categories & reviews)  

### Backend (Upcoming)
- Python Django  
- SQL Database (PostgreSQL/MySQL)  

### Deployment
- Vercel (frontend live demo)
